# Content Calendar & Daily Status Reporter

A UI-focused content planning dashboard for digital marketing teams. Plan posts, track status by client, and preview a daily report email.

## Highlights

- Week and day-based calendar layouts
- Status-aware client cards (Draft, Scheduled, Posted, Missed)
- Inline add/edit/remove flows with confirmation
- Daily report dialog with prefilled recipient
- Light/dark mode toggle

## Getting Started

```bash
npm install
npm run dev
```

## Project Structure

- `src/pages` - page-level layouts
- `src/components` - reusable UI and feature components
- `src/constants` - seeded demo data
- `src/types` - shared types
